## Description

A versatile mobile robot platform built for research in autonomous navigation and human-robot interaction.

## Components

- Custom differential drive chassis
- RPLiDAR A1 2D laser scanner
- Intel RealSense D435i depth camera
- Raspberry Pi 4 + Arduino Mega
- Custom battery management system

## Capabilities

- Autonomous navigation with Nav2
- 2D SLAM with Cartographer
- Object detection and tracking
- Teleoperation interface

## Applications

Used for teaching robotics courses and as a testbed for navigation algorithms.
